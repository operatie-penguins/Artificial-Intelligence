Championship League Tactical Simulator Readme
=============================================

Установка и запуск игры:

1. Разархивируйте игру.

2. Зарегистрируйте в системе библиотеку MSDATGRD.OCX,
   для этого наажмите Start->Run, в появившейся строке наберите:
   regsvr32 /i Ваша_папка_с_игрой\SIMPR\MSDATGRD.OCX

   Например, если Вы разархивировали игру в папку C:\Games\CLTS,
   то данная стркоа будет выглядеть так:
   regsvr32 /i C:\Games\CLTS\SIMPR\MSDATGRD.OCX		  
   Нажмите ОК. Если возникнет окно с предупреждением - еще раз ОК.

3. Зарегистрируйте в системе библиотеку MSFLXGRD.OCX
   Для этого повторите предыдущую операцию, но теперь введите строку:
   regsvr32 /i Ваша_папка_с_игрой\SIMPR\MSFLXGRD.OCX

4. Запутсите CLTS_1.0.smp двойным щелчком мыши и выберите на панели
   кнопку "Запуск" (черный треугольник в зеленой рамке) или выберите
   в меню Выполнение->Запуск.
   
   Предупреждение: Внесение каких-либо изменений в файл CLTS_1.0.smp
   может повелчь за собой сбои в работе программы.
